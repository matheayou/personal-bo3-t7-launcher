# BO3 T7 Launcher
A private quality-of-life helper tool for Call of Duty: Black Ops III.

## How to Install
1. Double-click the installer `TSetup.exe` and follow the on-screen steps.
2. If Steam launch options do not apply automatically, refer to the manual instructions in the install directory.

## Features
* Securely boots T7 Patch and stabilizes your game startup.
* Automatically closes companion tools when you exit the game.
